# Financial BOT LLM

## Overview
An AI-powered financial assistant that uses Retrieval-Augmented Generation (RAG) to answer questions from financial documents using OpenAI models and vector search.

## Features
- Document ingestion
- Vector embeddings
- RAG pipeline
- Conversational financial assistant
- Source citation
- OpenAI GPT integration

## Technologies
- Python
- OpenAI API
- ChromaDB (or your vector database)
- LangChain (if used)
- python-dotenv
